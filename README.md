# Watchstore
Backend e-commerce platform made with modern .NET and Azure Cloud technologies.

## Features
- Azure IaC deployment with .NET Aspire
- Local authentication with Keycloak
- Cloud authentication with Microsoft Entra
- CI/CD pipeline with Azure Devops
- Integration Tests
- Outbox pattern and Service bus for safer payments
- Stripe integration

## Tech Stack
- .NET 10
- ASP.NET & EF Core
- .NET Aspire Apphost Orchestration
- PostgreSQL
- Azure Storage (Blobs) 
- Azure Sevrice Bus
- Azure Key Vault
- Azure App Insights

## Setup
Required to work:
- .NET 10 Runtime
- A docker Runtime

## Demo
Currently Hosted at https://watchstore.azurewebsites.net

## Architecture
- ASP.NET Core Minimal APIs
- Vertical Slice Architecture
